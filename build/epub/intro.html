Introduction

Base URL

All URLs referenced in the documentation have the following base

https://api.opentact.org/v1/

The Opentact REST API is served over HTTPS. To ensure data privacy, unencrypted HTTP is not supported.

HTTP Base Authenication

The Opentact REST API is using HTTP Base Authenication, Username is sid, Password is your auth_token.
